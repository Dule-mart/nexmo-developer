# Configure the ASP.NET App

Open your `appsettings.json` file and add the following two fields to it:

```text
VONAGE_API_KEY
VONAGE_API_SECRET
```

Add the API key and API secret found in your [Dashboard](https://dashboard.nexmo.com/)
